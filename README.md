### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/RFer7935/Dungeon-Tales-.git
   ```

2. Open the project in Godot:
   - Launch Godot Engine
   - Click "Import"
   - Navigate to the cloned directory
   - Select `project.godot`
   - Click "Import & Edit"

   - Press F5 or click the Play button in Godot
